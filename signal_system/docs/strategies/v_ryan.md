# Ryan 精简 CANSLIM 策略说明

> **策略ID**: `v_ryan`
> **灵感来源**: David Ryan（O'Neil 学生，3届美国投资大赛冠军）
> **适用场景**: 极致精选，宁缺毋滥

---

## 📚 策略哲学

**核心理念**："只在形态完美时出手，极紧 VCP + 极紧止损。"

David Ryan 是 O'Neil 的学生，曾连续三年获得美国投资锦标赛冠军。他的特点是：
- 极其挑剔的选股标准
- 只买形态最紧凑的股票
- 快速止损（8%），快速止盈（15%）

**策略效果**：信号极少（0.6个/月），但每次出手都是精品。

---

## 🎯 核心特点（与其他策略的差异）

### 1. 极紧 VCP（最严格）

```
v1_plus: VCP 末端箱体 < 12%
v_oneil: VCP 末端箱体 < 20%
v_ryan:  VCP 末端箱体 < 5%   ← 极致严格
```

**什么是"极紧 VCP"？**

```
股价走势：
高点1 ───┐
         │ 回调 6%
高点2 ───┤
         │ 回调 3%
高点3 ───┤
         │ 回调 0.5%（几乎横盘）
突破 ─────┘

末端箱体 < 5% = 最近10天价格波动极小
```

**为什么这么紧？**
- Ryan 发现：形态越紧凑，突破后爆发力越强
- 极紧箱体 = 供需极度平衡，一触即发
- 只有最完美的形态才买入

---

### 2. 极紧追踪止损

```
v1_plus: 从高点回落 20% 止盈
v_oneil: 从高点回落 20% 止盈
v_ryan:  从高点回落 15% 止盈  ← 更紧
```

**逻辑**：
- 快速锁定利润
- 不让大利润回吐
- 如果从高点回落 15%，说明趋势可能走弱

---

### 3. 放量要求更高

```
v1_plus: 成交量 ≥ 1.5 倍均量
v_ryan:  成交量 ≥ 2.0 倍均量  ← 更严格
```

**为什么要 2 倍放量？**
- Ryan 强调：真正的突破必须有机构大资金入场
- 1.5 倍可能只是散户追涨
- 2 倍以上说明大资金认可

---

### 4. RS 排名更高

```
v1_plus: RS ≥ 70%（前30%）
v_ryan:  RS ≥ 80%（前20%）
```

与 O'Neil 相同，只买领涨股。

---

## 🎯 买入条件

### 必须同时满足（缺一不可）

1. **趋势确认**
   - SMA50 > SMA150 > SMA200
   - SMA200 上升
   - 距 52 周低点 ≥ +25%
   - 距 52 周高点 ≤ 25%

2. **RS 排名 ≥ 80%**
   - 过去 12 个月涨幅排名前 20%

3. **极紧 VCP**
   - 至少 2 次波动收缩
   - **末端箱体 < 5%**（最严格）
   - 量能萎缩

4. **放量突破**
   - 突破过去 30 日最高价
   - **成交量 ≥ 2.0 倍均量**

## 🚪 卖出条件

| 条件 | 触发标准 | 说明 |
|------|---------|------|
| **固定止损** | 买入价 × 92% | 亏损 8% 立即退出 |
| **追踪止盈** | 最高价 × 85% | 从高点回落 15% |
| **时间止损** | 20天且盈利<3% | 快速周转 |

**Ryan 的纪律**：
- 买入后立即设置 8% 止损
- 涨到一定幅度后，将止损移至成本价（保本）
- 高点回撤 15% 坚决退出

## 🎓 策略优缺点

### ✅ 优点

1. **精选优质股**：每次出手都是形态最完美的股票
2. **风险可控**：15% 追踪止盈保护利润
3. **胜率可能高**：形态越完美，成功率越高
4. **符合大师实践**：David Ryan 三届冠军的实战验证

### ⚠️ 缺点

1. **信号极少**：OOS 期间仅 14 笔交易（0.6个/月）
2. **无法分散风险**：信号太少，单笔影响大
3. **样本外表现差**：
   - 年化收益仅 14.0%
   - 胜率 21.4%
   - 夏普比率 0.61
   - **综合评级 1/6（不合格）**
4. **条件过严**：错失大量机会

## 💡 为什么样本外表现差？

### 假设分析

1. **条件过严导致错失机会**
   ```
   OOS 期间（2024-2026）市场特征：
   - 牛市主升浪，很多股票形态不完美但涨势良好
   - 极紧 VCP（< 5%）出现频率低
   - 错过了很多"中等形态"的优质股
   ```

2. **胜率悖论**
   ```
   理论上：形态越完美 → 胜率越高
   实际上：样本外胜率仅 21.4%

   可能原因：
   - 完美形态可能被"做图"（主力诱多）
   - VCP < 5% 在牛市中过于罕见
   - 14 笔交易样本太小，统计意义不足
   ```

3. **时机问题**
   ```
   Ryan 在 1980s 使用此策略 → 不同市场环境
   2024-2026 牛市 → 极紧形态反而少见
   可能更适合震荡市
   ```

## 💡 改进建议

### 1. 放宽 VCP 末端箱体

```
当前：< 5%
建议：< 8%（接近 v1_plus）

理由：
- v1_plus 的 12% 效果更好
- 5% 过于严格，错失机会
```

### 2. 降低放量要求

```
当前：≥ 2.0 倍
建议：≥ 1.5 倍

理由：
- 2.0 倍在牛市中过于严格
- 1.5 倍已足够确认机构入场
```

### 3. 降低 RS 门槛

```
当前：≥ 80%
建议：≥ 75%

理由：
- 80% 排除太多股票
- 75% 仍在前四分位
```

### 4. 调整追踪止盈

```
当前：15%
建议：18-20%

理由：
- 15% 太紧，容易被震出
- 成长股正常波动 15-20%
```
