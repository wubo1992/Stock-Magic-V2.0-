# Kullamägi VCP 变种策略说明

> **策略ID**: `v_kullamaggi`
> **灵感来源**: Kristjan Kullamägi（Qullamaggie，欧洲顶级交易员）
> **适用场景**: 极紧止损，极端放量，Episodic Pivot（财报惊喜）

---

## 📚 策略哲学

**核心理念**："用最紧的止损，捕捉最极端的放量突破。"

Kristjan Kullamägi（网名 Qullamaggie）是瑞典交易员，以极紧止损（5%）和极端放量（4倍）著称。他的特点：
- 止损 5%（最严格）
- 放量 4 倍（最极端）
- 追求 Episodic Pivot（财报超预期后的突破）

**核心方法**：在财报超预期的当天，如果股票放量突破，立即买入。这是机构抢筹的时刻。

---

## ⚠️ 重要限制

**本策略未实现 Episodic Pivot**（财报超预期），原因：
- 缺少财报发布日期数据
- 缺少 EPS 超预期百分比数据
- 缺少分析师预期数据

**当前实现**：仅技术面（极紧止损 + 极端放量 + VCP）

**影响**：信号质量低于 Kullamägi 原版。

---

## 🎯 核心特点（与其他策略的差异）

### 1. 最紧止损（5%）

```
v_oneil: 止损 7%
v_ryan:  止损 8%
v_kell:  止损 8%
v_kullamaggi: 止损 5%  ← 最严格
```

**为什么要 5%？**
- Kullamägi 发现：健康股票突破后很少回调超过 5%
- 跌破 5% = 突破失败或市场转弱
- 快速止损，快速重新入场

**风险**：
- 容易被正常波动震出
- 需要更精确的入场时机

---

### 2. 最极端放量（4 倍）

```
v1_plus: 成交量 ≥ 1.5 倍
v_ryan:  成交量 ≥ 2.0 倍
v_kell:  成交量 ≥ 3.0 倍
v_kullamaggi: 成交量 ≥ 4.0 倍  ← 极端
```

**为什么是 4 倍？**
- Kullamägi 认为只有极端放量才值得信任
- 4 倍放量通常出现在：
  - 财报超预期
  - 重大利好消息
  - 机构大规模建仓

---

### 3. 追踪止盈更紧（12%）

```
v1_plus: 从高点回落 20%
v_ryan:  从高点回落 15%
v_kullamaggi: 从高点回落 12%  ← 更紧
```

**逻辑**：
- 快速锁定利润
- 不让盈利变亏损
- 极紧风格贯穿始终

---

### 4. 时间止损更短（15 天）

```
v1_plus: 持仓 20 天
v_kullamaggi: 持仓 15 天  ← 更短
```

**为什么？**
- Episodic Pivot 突破后应快速上涨
- 15 天不涨 = 突破失败
- 资金快速周转

## 🎯 买入条件

### 必须同时满足

1. **趋势确认**
   - SMA50 > SMA150 > SMA200
   - SMA200 上升
   - 距 52 周低点 ≥ +25%
   - 距 52 周高点 ≤ 25%

2. **RS 排名 ≥ 75%**
   - 稍低于 O'Neil 的 80%（更宽松）

3. **VCP 形态**
   - 至少 2 次波动收缩
   - 末端箱体 < 8%
   - 量能萎缩

4. **突破**
   - 突破过去 30 日最高价
   - 突破幅度 > 0.5%

5. **极端放量**
   - **成交量 ≥ 4.0 倍均量**（最严格）

## 🎓 策略优缺点

### ✅ 优点

1. **风险控制极严**：5% 止损 + 12% 追踪止盈
2. **最大回撤最低**：13.9%（所有策略中最低）
3. **夏普比率优秀**：1.96
4. **符合冠军方法**：Kullamägi 实战验证

### ⚠️ 缺点

1. **信号极少**：0.4 个/月（远低于标准）
2. **缺少 Episodic Pivot**：无财报数据，质量下降
3. **容易被震出**：5% 止损容易被正常波动触发
4. **样本外交易太少**：仅 10 笔交易（统计意义不足）

## 💡 Episodic Pivot 缺失的影响

**Kullamägi 完整方法**：
```
Episodic Pivot = 财报超预期 + 放量突破

示例：
公司发布财报：
- EPS 实际：$1.50
- 分析师预期：$1.20
- 超预期：25%

股价反应：
- 当日跳空高开 +5%
- 成交量暴增 4-5 倍
- 机构抢筹

Kullamägi 策略：
- 在开盘后第一时间买入
- 止损 5%
- 通常能在几天内获利 10-20%
```

**我们缺失的部分**：
```
当前实现：
- 只检测技术突破
- 无财报日期信息
- 无 EPS 超预期数据

影响：
- 错过真正的 Episodic Pivot 机会
- 信号质量下降
- 假突破概率上升
```

## 💡 实战建议

### 1. 结合财报日历（如果可能）

```
完整策略需要：
1. 获取财报发布日期（如 Zacks, Earnings Whispers）
2. 获取 EPS 超预期百分比
3. 在财报发布后第一天，检测是否有 4 倍放量突破
```

### 2. 止损纪律

```
Kullamägi 的建议：
"5% 止损是铁律，不要给股票第二次机会"
"被震出后，如果股票重新走强，可以重新入场"

实际操作：
- 买入时设置 5% 止损单
- 被震出后，观察是否重新放量突破
- 可以重新入场，但止损仍为 5%
```

### 3. 接受信号极少

```
v_kullamaggi 每月 0.4 个信号 = 每 2.5 个月 1 个信号

心理准备：
- 可能连续 2-3 个月无信号
- 一旦出现信号，质量极高
- 4 倍放量突破很难造假
```
