# Stop Loss Plus 策略回测报告

## 一、策略逻辑

### 1.1 策略名称
Stop Loss Plus（止损增强版）

### 1.2 策略规则

1. **买入条件**
   - 任何现有策略产生买入信号（9大策略：v1, v1_plus, v_oneil, v_ryan, v_kell, v_kullamaggi, v_zanger, v_stine, v_weinstein）
   - 信号产生的次日开盘买入
   - 每次买入金额：$1,000

2. **卖出条件（一级制）**
   - **止盈**：盈利 +20% 时卖出
   - **止损**：亏损 -10% 时卖出

3. **持仓周期**
   - 不设时间限制，持续持仓直到触发止盈或止损

### 1.3 策略思想

- **让利润奔跑**：不急于止盈，让盈利充分发展
- **截断亏损**：严格止损，避免更大的损失
- **盈亏比优势**：通过 2:1 的盈亏比（止盈20% vs 止损10%），即使胜率低于50%也能盈利

## 五、数学原理

### 5.1 为什么低胜率也能盈利？

核心在于**盈亏比**：

| 盈亏比 | 盈亏平衡胜率 |
|--------|-------------|
| 1:1 | 50% |
| 2:1 | 33.3% |
| 3:1 | 25% |

当盈亏比为 2:1 时，只需要 **34%** 的胜率就能保本，超过这个比例就开始盈利。

### 5.2 为什么两级版本不如一级版本？

两级版本的问题：
- **+10% 卖一半**：过早锁定部分利润，错过后续涨幅
- **-5% 卖一半**：亏损后仍保留一半仓位，可能造成更大损失

一级版本的优势：
- 让利润充分奔跑（等到 +20%）
- 果断截断亏损（-10% 全部卖出）
- 更少的交易次数 = 更低的摩擦成本

4. ✅ 数学期望为正，长期执行可期待盈利

**核心理念**：让利润奔跑，截断亏损。
