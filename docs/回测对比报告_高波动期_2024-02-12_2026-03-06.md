#### 信号过多（>10/月）

**v_zanger（13.1/月）**
- 原因：无 RS、无 VCP 过滤
- 建议：提高放量至 4.0x 或添加 RS ≥ 60% 过滤

#### 信号过少（<1/月）

**v_stine（0.1/月）**
- 原因：RS≥90% + VCP<5% + 放量 3x，条件过严
- 建议：作为质量评分器，不单独使用

**v_kullamaggi（0.4/月）**
- 原因：放量 4x + 止损 5%，条件极严
- 建议：放宽放量至 3.5x 或止损至 6%

**v_kell（0.9/月）**
- 原因：放量 3x + 突破≥1.5%
- 建议：与其他策略组合使用

**v_ryan（0.6/月）**
- 原因：VCP <5%，条件过严
- 建议：放宽 VCP 至 7%

### 4. 回撤控制最佳：v_kullamaggi（13.9%）

**v_kullamaggi 特点**：
- 极紧止损：5%
- 极紧追踪止盈：12%
- 极端放量：4x
- 时间止损：15 天

**为什么回撤最低？**
- 快速止损（5%），不让亏损扩大
- 快速锁定利润（12%），不让盈利变亏损
- 高质量筛选（4x 放量），减少假突破

**劣势**：
- 信号极少（0.4/月）
- 容易被正常波动震出

---

### 5. RS 排名 ≠ 收益率

```
RS≥90%（v_stine）：   51.4% 年化
RS≥80%（v_oneil）：   96.1% 年化
RS≥80%（v_ryan）：    14.0% 年化  ← 失败案例
RS≥70%（v1_plus）：   45.3% 年化
无 RS（v_zanger）：    128.4% 年化 ← 最佳
```

**结论**：
- RS 只是筛选器，不是收益保证
- RS 过严会导致信号过少
- 无 RS 也能高收益（v_zanger）

---

### 6. VCP 严格度 ≠ 收益率

```
VCP <5%（v_ryan）：    14.0% 年化  ← 失败案例
VCP <5%（v_stine）：   51.4% 年化
VCP <8%（v_kullamaggi）： 50.6% 年化
VCP <10%（v_kell）：   80.2% 年化
VCP <12%（v1_plus）：  45.3% 年化
VCP <20%（v_oneil）：  96.1% 年化
无 VCP（v_zanger）：    128.4% 年化 ← 最佳
```

**结论**：
- VCP 太紧会导致信号过少
- VCP 宽松也能高收益（v_oneil）
- 无 VCP 也能高收益（v_zanger）

---

### 7. 放量要求越高 → 信号越少 → 质量不一定更高

```
放量 4.0x（v_kullamaggi）： 0.4 个/月，50.6% 年化
放量 3.0x（v_kell）：        0.9 个/月，80.2% 年化
放量 3.0x（v_zanger）：      13.1 个/月，128.4% 年化
放量 1.5x（v1_plus）：       6.4 个/月，45.3% 年化
```

**结论**：
- 放量要求需平衡信号频率和质量
- 放量过高导致信号过少
- 放量 3x 可以高收益（v_zanger）也可以低频（v_kell）
