## 🎯 策略分类

### 按信号频率

#### 高频策略（>10 信号/月）

**v_weinstein（28.5/月）**
- 特点：只用 SMA150 趋势 + 突破，条件极简
- 优点：样本外表现优秀（56.9% 年化，2.82 夏普）
- 缺点：信号过多，难以管理
- 建议：提高放量至 2.5x 控制频率

**v_zanger（13.1/月）**
- 特点：无 RS、无 VCP，纯技术突破 + 3x 放量
- 优点：样本外最佳（128.4% 年化，4.42 夏普，12% 回撤）
- 缺点：信号超标，可能追高
- 建议：提高放量至 4.0x 或添加 RS ≥ 60% 过滤

#### 标准频率策略（2-10 信号/月）

**v1_plus（6.8/月）** ✅ 基准策略
- 特点：平衡版，RS≥70%，VCP<12%，放量 1.5x
- 优点：唯一 6/6 全部达标策略
- 缺点：无（平衡性最佳）
- 建议：作为核心底仓（50% 资金）

**v_oneil（3.2/月）**
- 特点：宽松 VCP（<20%），RS≥80%，止损 7%
- 优点：信号质量较高，胜率 42.9%
- 缺点：收益略低于 v1_plus
- 建议：与 v1_plus 组合使用

#### 低频策略（<2 信号/月）

**v_kell（0.9/月）**
- 特点：极端放量（3x），强势突破（≥1.5%）
- 优点：高盈亏比（4.43），高收益（80.2%）
- 缺点：回撤大（29.4%），胜率低（18.2%），样本小（22 笔）
- 建议：与低波动策略组合

**v_ryan（0.6/月）**
- 特点：极紧 VCP（<5%），RS≥80%
- 优点：条件严格，形态完美
- 缺点：样本外表现差（14% 年化），样本小（15 笔）
- 建议：放宽 VCP 至 7% 增加信号

**v_kullamaggi（0.4/月）**
- 特点：极紧止损（5%），极端放量（4x）
- 优点：回撤最低（13.9%），风险控制最佳
- 缺点：信号极少，缺少 Episodic Pivot 数据
- 建议：放宽放量至 3.5x 或止损至 6%

**v_stine（0.1/月）**
- 特点：RS≥90%，极紧 VCP（<5%），放量 3x
- 优点：胜率 100%（仅 2 笔），质量最高
- 缺点：样本不足，缺少内部人交易数据
- 建议：作为质量评分器，不单独使用

## 🎯 核心差异对比

### RS 排名要求

```
v_stine:     RS ≥ 90%  ← Top 10%（最严格）
v_oneil:     RS ≥ 80%  ← Top 20%
v_ryan:      RS ≥ 80%  ← Top 20%
v_kell:      RS ≥ 80%  ← Top 20%
```

## 🔬 关键发现

### 1. RS 排名 ≠ 收益率

```
RS≥90%（v_stine）：51.4% 年化
RS≥80%（v_oneil）：46.4% 年化
RS≥80%（v_ryan）：14.0% 年化  ← 失败案例
无 RS（v_zanger）：128.4% 年化 ← 最佳
```

**结论**：RS 只是筛选器，不是收益保证。

---

### 2. VCP 严格度 ≠ 收益率

```
VCP <5%（v_ryan）：14.0% 年化  ← 失败案例
VCP <5%（v_stine）：51.4% 年化
VCP <10%（v_kell）：80.2% 年化
VCP <12%（v1_plus）：54.8% 年化
VCP <20%（v_oneil）：46.4% 年化
无 VCP（v_zanger）：128.4% 年化 ← 最佳
```

**结论**：VCP 太紧会导致信号过少，样本外表现不稳定。

---

### 3. 放量要求越高 → 信号越少 → 质量不一定更高

```
放量 4.0x（v_kullamaggi）：0.4 个/月，50.6% 年化
放量 3.0x（v_kell）：0.9 个/月，80.2% 年化
放量 3.0x（v_zanger）：13.1 个/月，128.4% 年化
放量 1.5x（v1_plus）：6.8 个/月，54.8% 年化
```

**结论**：放量要求需平衡信号频率和质量。

---

### 4. 止损越紧 → 回撤越小 → 但容易被震出

```
止损 5%（v_kullamaggi）：回撤 13.9%，信号 0.4/月
止损 6%（v_zanger）：回撤 12.0%，信号 13.1/月
止损 10%（v1_plus）：回撤 15.3%，信号 6.8/月
```

**结论**：止损宽度需配合信号频率。
