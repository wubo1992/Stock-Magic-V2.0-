# 魔法师调整参数版V1+ 策略说明

> **策略ID**: `v1_plus`
> **灵感来源**: Mark Minervini《股票魔法师》SEPA + VCP 策略
> **适用场景**: 牛市主升浪，趋势明确的市场环境

---

## 📚 策略哲学

**核心理念**："只在股价经过充分整理、能量高度压缩后，放量突破时买入。"

想象一个弹簧：
- 股价反复震荡、成交量逐渐萎缩 = 弹簧被压紧
- 突然放量突破前高 = 弹簧释放，爆发力最强
- 我们的策略就是捕捉这个"释放瞬间"

---

## 🎯 买入条件（缺一不可）

### 1️⃣ 趋势确认（8个条件）

**均线多头排列**
```
股价位置：在所有均线之上
├─ SMA50（短期）> SMA150（中期）> SMA200（长期）
├─ SMA200 本身正在上升（斜率向上）
└─ 三条线从上到下依次排列，像阶梯一样
```

**52周高低点过滤**
```
当前位置：
├─ 距离52周最低点至少 +25%（已经脱离底部）
└─ 距离52周最高点不超过 25%（仍在高位区）
```

**通俗解释**：
- 股票已经涨了一段（不是底部抄底）
- 但还没涨到天花板（不是追高）
- 三条均线都向上，说明短期、中期、长期趋势一致

---

### 2️⃣ 相对强度 RS（Relative Strength）

**条件**：过去12个月涨幅排名 ≥ 前30%

```
示例：市场上有100只股票
你的股票涨幅要排在前30名以内
```

**为什么重要**：
- RS 排名高的股票 = 市场上的"优等生"
- 牛市中强者恒强，龙头股涨幅远超大盘
- 避免"跟涨不跟跌"的弱势股

---

### 3️⃣ VCP 形态识别（核心！）

**VCP = Volatility Contraction Pattern（波动收缩形态）**

```
股价走势：
高点1 ──┐
        │ 回调 8%
高点2 ──┤（比高点1低）
        │ 回调 4%（收缩！）
高点3 ──┤（比高点2低）
        │ 回调 0%（极度收缩！）
突破 ───┘
```

**关键特征**：
1. **波动递减**：每次回调幅度都比上一次小
   - 第1次回调：-8%
   - 第2次回调：-4%
   - 第3次回调：-2%（越来越小）

2. **量能萎缩**：每次回调期间的成交量逐渐减少
   - 说明卖盘在枯竭
   - 持有者不愿意卖，供给减少

3. **末端箱体**：最近10天价格波动 < 12%
   - 就像弹簧压紧到最后
   - 能量极度蓄积

**v1_plus 的调整**：
- 原版 VCP 末端箱体要求 < 8%
- v1_plus 放宽到 < 12%
- 为什么？历史回测发现，适度放宽能捕获更多机会，收益反而更高

---

### 4️⃣ 枢轴点突破

**条件**：今日收盘价 > 过去30日最高收盘价 × 1.005

```
示例：
过去30天最高价 = $100
今日收盘价必须 ≥ $100.5（至少突破0.5%）
```

**为什么是30日？**
- 太短（如10日）：容易被假突破
- 太长（如60日）：错过最佳入场时机
- 30日是平衡点

**突破幅度评估**：
- 0.5%-1.5%：普通突破
- 1.5%-3.0%：较强突破
- > 3.0%：强势突破（可能追高）

---

### 5️⃣ 成交量确认

**条件**：今日成交量 ≥ 20日均量 × 1.5

```
示例：
过去20天平均成交量 = 100万股
今日成交量必须 ≥ 150万股
```

**为什么重要**：
- 放量 = 大资金真实入场
- 缩量突破 = 可靠性低，可能是假动作
- "量在价先"，成交量是机构资金的脚印

## 🚪 卖出条件（三重保护）

### 1. 固定止损（亏损保护）

```
买入价 × 90% = 止损价

示例：
买入价 $100
止损价 $90
跌破 $90 立即卖出，亏损控制在 10% 以内
```

**为什么10%？**
- Minervini 统计：健康股票回调很少超过 10%
- 跌破 10% = 趋势可能反转
- 宁可小亏，不可大亏

---

### 2. 追踪止盈（锁定利润）

```
持仓期间最高价 × 80% = 追踪止损价

示例：
买入价 $100
涨到 $150（最高价）
追踪止损价 = $150 × 80% = $120
股价跌回 $120 时卖出，锁定 $20 利润
```

**逻辑**：
- 让利润奔跑
- 但保护住主要利润
- 从高点回落 20% = 趋势可能走弱

---

### 3. 时间止损（资金效率）

```
持仓 ≥ 20天 且 盈利 < 3% → 卖出

示例：
买入后 20 天
价格只从 $100 涨到 $102（+2%）
→ 卖出，寻找下一个机会
```

**为什么？**
- 健康的突破应该在 1-2 周内快速上涨
- 横盘 20 天不涨 = 突破失败
- 资金有时间成本，不要死守不动的股票

## 🎓 策略优缺点

### ✅ 优点

1. **系统化程度高**：5个条件缺一不可，避免情绪化决策
2. **风控完善**：三重止损机制，最大亏损可控
3. **历史验证**：样本外（2024-2026）年化收益 54.8%，夏普比率 2.24
4. **适合趋势市场**：牛市中捕捉龙头股主升浪

### ⚠️ 缺点

1. **横盘市场失效**：震荡市中频繁触发止损
2. **信号频率波动大**：
   - 牛市中期：6-8个信号/月
   - 熊市：可能整月无信号
3. **需要耐心**：平均持仓 20-60 天
4. **错过底部反弹**：只买趋势确立的股票，不抄底

**适合**：
- 有一定投资经验（了解均线、成交量）
- 能接受 10% 单笔亏损
- 追求稳健收益（年化 20-50%）

- 无法承受 10% 回撤
- 期待暴富（年化 100%+）
- 没时间定期检查信号（至少每周一次）

## 🔍 常见问题

**Q1: 为什么放宽 VCP 末端箱体到 12%？**
A: 回测发现，原版 8% 过于严格，错过很多优质突破。12% 在保持形态有效性的同时，捕获更多机会。

**Q2: 信号出现后，第二天开盘买入还是收盘买入？**
A: 开盘买入。Minervini 强调"突破次日开盘即买入"，不要等收盘确认。

**Q3: 止损触发后，股票又涨了怎么办？**
A: 这是正常的。接受"卖飞"是策略的一部分。宁可错过利润，不可承受大亏。止损触发后可以重新入场。

**Q4: 能否同时运行多个策略？**
A: 可以。建议 v1_plus 为主（70%仓位），v_stine 为辅（30%仓位），分散风险。
