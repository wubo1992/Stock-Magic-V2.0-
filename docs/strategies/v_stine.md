# Stine 超强精选策略说明

> **策略ID**: `v_stine`
> **灵感来源**: Jesse Stine（《超级周期》作者）
> **适用场景**: 极端精选，Top 10% 领涨股

---

## 📚 策略哲学

**核心理念**："只买最强的 10%，只有最好的股票才值得持有。"

Jesse Stine 的方法：
- RS 排名 ≥ 90%（Top 10%）
- 极紧 VCP（5% 箱体）
- 内部人士买入（SEC Form 4）
- 基本面 + 技术面 + 内部人共振

**核心逻辑**：如果一只股票在所有维度都完美（相对强度、形态、基本面、内部人），失败概率极低。

---

## ⚠️ 重要限制

**本策略未实现内部人士买入条件**，原因：
- 缺少 SEC Form 4 数据（内部人交易披露）
- 需要付费数据源（如 OpenInsider, InsiderCow）
- 这是 Stine 策略的核心过滤器之一

**影响**：信号质量低于原版，建议谨慎使用。

---

## 🎯 核心特点（与其他策略的差异）

### 1. 最高 RS 门槛

```
v1_plus: RS ≥ 70%（前30%）
v_oneil: RS ≥ 80%（前20%）
v_ryan:  RS ≥ 80%（前20%）
v_stine: RS ≥ 90%（前10%）← 最严格
```

**Stine 的逻辑**：
- 过去 12 个月涨幅排名前 10%
- 这些是市场的绝对龙头
- 强者恒强，龙头涨幅远超大盘

**数据支持**：
- IBD 研究显示：RS≥90 的股票平均涨幅是 RS<50 的 3-5 倍

---

### 2. 极紧 VCP（与 Ryan 相同）

```
v1_plus: VCP 末端 < 12%
v_stine: VCP 末端 < 5%  ← 与 v_ryan 相同
```

**为什么极紧？**
- Stine 要求形态近乎完美
- 末端箱体 < 5% = 极度压缩
- 突破后爆发力最强

---

### 3. 极端放量（与 Kell 相同）

```
v1_plus: 成交量 ≥ 1.5 倍
v_stine: 成交量 ≥ 3.0 倍  ← 与 v_kell 相同
```

**3 倍放量的意义**：
- 机构大资金入场
- 与 Kell 相同的标准
- 确保突破质量

---

### 4. 接近历史高点

```
v1_plus: 距 52 周高点 ≤ 25%
v_stine: 距 52 周高点 ≤ 25%（相同）
```

Stine 要求股票已经接近或创新高，不抄底。

## 🎯 买入条件

### 必须同时满足（缺一不可）

1. **趋势确认**
   - SMA50 > SMA150 > SMA200
   - SMA200 上升
   - 距 52 周低点 ≥ +25%
   - 距 52 周高点 ≤ 25%

2. **RS 排名 ≥ 90%**
   - **过去 12 个月涨幅排名前 10%**（最严格）
   - 只买市场最强股票

3. **极紧 VCP**
   - 至少 2 次波动收缩
   - **末端箱体 < 5%**（最严格）
   - 量能萎缩

4. **放量突破**
   - 突破过去 30 日最高价
   - **成交量 ≥ 3.0 倍均量**

5. **内部人士买入（原版要求）**
   - ❌ **未实现**：缺少 SEC Form 4 数据

## 💡 为什么信号这么少？

### 三个最严格条件的交集

```
条件 1: RS ≥ 90%（Top 10%）
        → 从 300 只股票筛选到 30 只

条件 2: VCP 末端 < 5%
        → 从 30 只筛选到 3 只

条件 3: 放量 ≥ 3 倍
        → 从 3 只筛选到 0.1 只/月

结果：平均每 10 个月才有 1 次买入机会
```

**OOS 期间（517 天）**：
- 只有 2 笔交易
- 平均每 258 天才出现 1 次信号
