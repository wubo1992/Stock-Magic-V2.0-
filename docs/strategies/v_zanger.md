# Zanger 纯技术动量策略说明

> **策略ID**: `v_zanger`
> **灵感来源**: Dan Zanger（世界纪录保持者，18个月 29,000% 收益率）
> **适用场景**: 纯技术突破，牛市趋势股

---

## 📚 策略哲学

**核心理念**："价格突破是唯一的真理，成交量大资金说了算。"

Dan Zanger 在 1998-1999 年创造了 29,000% 的收益率世界纪录。他的特点是：
- 不看基本面（无 RS、无 VCP）
- 只看技术突破
- 成交量是唯一验证
- 快进快出，10 天时间止损

**核心方法**：当价格突破整理区且成交量暴增 3 倍以上，立即买入，快速止损（6%）。

---

## 🎯 核心特点（与其他策略的差异）

### 1. 无相对强度 RS

```
v1_plus: RS ≥ 70%（必须有）
v_zanger: 无 RS 过滤  ← 完全不看
```

**为什么不用 RS？**
- Zanger 认为价格突破本身已经证明股票强度
- RS 是历史表现，突破是当下动能
- 过滤 RS 可能错过新兴牛股

---

### 2. 无 VCP 形态

```
v1_plus: 必须有 VCP 形态
v_zanger: 无 VCP 检测  ← 完全不看形态
```

**为什么不用 VCP？**
- Zanger 更关注突破时的动能，而非整理形态
- 整理形态完美与否不是关键
- 放量突破才是确认信号

---

### 3. 只用 SMA150

```
v1_plus: SMA50 > SMA150 > SMA200（三条均线）
v_zanger: SMA150 上升（一条均线）← 极简化
```

**逻辑**：
- SMA150 ≈ 30 周均线（Weinstein 也用）
- 只要 SMA150 向上，说明中期趋势向上
- 不需要短期均线确认

---

### 4. 极端放量 3 倍

```
v1_plus: 成交量 ≥ 1.5 倍
v_zanger: 成交量 ≥ 3.0 倍  ← 与 Kell 相同
```

**为什么是 3 倍？**
- Zanger 的经验：真正的突破必须有大资金入场
- 3 倍放量是机构入场的标志
- 缩量突破不可靠

---

### 5. 最短时间止损（10 天）

```
v1_plus: 20 天
v_kullamaggi: 15 天
v_zanger: 10 天  ← 最短
```

**Zanger 的逻辑**：
- 如果突破是真实的，应该在 10 天内上涨
- 横盘 10 天不涨 = 突破失败
- 资金快速周转，不要等待

---

### 6. 更紧止损（6%）

```
v_kullamaggi: 5%
v_zanger: 6%
v_oneil: 7%
```

Zanger 的止损纪律：6% 是平衡点，既能容忍小幅波动，又能快速止损。

## 🎯 买入条件（极简化）

### 只需 4 个条件

1. **SMA150 上升**
   ```
   当前 SMA150 > 20 天前 SMA150
   说明：中期趋势向上
   ```

2. **价格在 SMA150 之上**
   ```
   当前价格 > SMA150
   说明：股价在趋势上方
   ```

3. **突破 30 日高点**
   ```
   当前价格 > 过去 30 日最高价 × 1.005
   说明：创近期新高
   ```

4. **放量 3 倍**
   ```
   当前成交量 ≥ 20 日均量 × 3.0
   说明：大资金入场
   ```

**就这么简单！** 无 RS、无 VCP、无 52 周高低点过滤。

## 🎓 策略优缺点

### ✅ 优点

1. **极简主义**：只看 4 个条件，易于理解和执行
2. **样本外表现最佳**：年化收益 128.4%，夏普 4.42，回撤 12.0%
3. **无基本面依赖**：不需要 EPS、ROE 等数据
4. **捕捉动量**：专注突破时的动能

### ⚠️ 缺点

1. **信号过多**：13.1 个/月（超过 10 个标准）
2. **可能追高**：无 RS、无 VCP，容易买到假突破
3. **快速周转**：10 天时间止损，交易频率高
4. **不适合震荡市**：横盘市会频繁止损
