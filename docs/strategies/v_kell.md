# Kell VCP 2.0 策略说明

> **策略ID**: `v_kell`
> **灵感来源**: Oliver Kell（2020年美国投资锦标赛冠军，+941%）
> **适用场景**: 极端放量确认，牛市趋势股

---

## 📚 策略哲学

**核心理念**："只有极端放量才值得信任，突破必须强势。"

Oliver Kell 在 2020 年以 941% 的收益率赢得美国投资锦标赛。他的特点是：
- 要求极端放量（3倍以上）
- 要求强势突破（至少 1.5%）
- 宁缺毋滥，只做最有把握的

**注意**：本策略缺少 EPS≥50% 基本面筛选。

---

## 🎯 核心特点（与其他策略的差异）

### 1. 极端放量要求（最严格）

```
v1_plus: 成交量 ≥ 1.5 倍均量
v_ryan:  成交量 ≥ 2.0 倍均量
v_kell:  成交量 ≥ 3.0 倍均量  ← 极端严格
```

**为什么是 3 倍？**
```
普通放量（1.5倍）：散户追涨
中度放量（2.0倍）：部分机构入场
极端放量（3.0倍）：大机构重仓买入 ← Kell 只信这个
```

**极端放量的意义**：
- 机构资金入场（共同基金、对冲基金）
- 真正的供需失衡
- 突破的可靠性更高

---

### 2. 强势突破要求

```
v1_plus: 突破幅度 > 0.5%
v_kell:  突破幅度 > 1.5%  ← 3倍标准
```

**为什么要求 1.5%？**
- 0.5% 可能是日内波动
- 1.0% 仍不够强
- 1.5%+ 说明突破动能充足

---

### 3. VCP 适中

```
v_ryan:  VCP 末端 < 5%（极紧）
v_kell:  VCP 末端 < 10%（适中）
v_oneil: VCP 末端 < 20%（宽松）
```

Kell 的 VCP 要求介于 Ryan 和 O'Neil 之间，既不过于严苛，也不过于宽松。

---

### 4. RS 排名高标准

```
v_kell: RS ≥ 80%（与 O'Neil、Ryan 相同）
```

只买领涨股，只买市场认可度高的股票。

---

## 🎯 买入条件

### 必须同时满足

1. **趋势确认**
   - SMA50 > SMA150 > SMA200
   - SMA200 上升
   - 距 52 周低点 ≥ +25%
   - 距 52 周高点 ≤ 25%

2. **RS 排名 ≥ 80%**
   - 过去 12 个月涨幅排名前 20%

3. **VCP 形态**
   - 至少 2 次波动收缩
   - 末端箱体 < 10%
   - 量能萎缩

4. **强势突破**
   - 突破过去 30 日最高价
   - **突破幅度 ≥ 1.5%**（最严格）

5. **极端放量**
   - **成交量 ≥ 3.0 倍均量**（最严格）

## 🎓 策略优缺点

### ✅ 优点

1. **信号质量高**：极端放量 + 强势突破 = 可靠性高
2. **冠军验证**：Oliver Kell 2020 年 +941% 实绩
3. **避免假突破**：3 倍放量 + 1.5% 突破很难造假
4. **盈亏比优秀**：样本外盈亏比 4.43（最高）

### ⚠️ 缺点

1. **信号频率低**：0.9 个/月（低于 2-10 标准）
2. **胜率偏低**：18.2%（样本外）
3. **回撤较大**：29.4%（超标）
4. **样本外年化**：80.2%，但未达标（3/6）

| 指标 | v_kell | v1_plus | 评价 |
|------|--------|---------|------|
| 年化收益 | **80.2%** | 54.8% | 较高 |

| 胜率 | **18.2%** | 43.7% | 很低 |
| 盈亏比 | **4.43** | 3.98 | 最高 |

**结论**：高收益、高回撤、低胜率、高盈亏比。典型的"大赢小输"策略。

## 💡 为什么胜率低但盈亏比高？

```
v_kell 的特点：
- 胜率：18.2%（每 5 次交易只有 1 次盈利）
- 盈亏比：4.43（盈利时赚大钱）

示例：
交易 1：-8%（止损）
交易 2：-8%（止损）
交易 3：-8%（止损）
交易 4：-8%（止损）
交易 5：+43.7%（大赢）

净结果：+11.7%（5笔交易总体盈利）
```

**Kell 的逻辑**：
- 用极端放量筛选，确保突破质量
- 虽然假突破仍会止损（-8%）
- 但真突破时涨幅巨大（+40%+）
- 盈亏比 4:1 可以接受低胜率

## 🎯 适用人群

**适合**：
- 能承受 30% 回撤
- 接受低胜率（20% 以下）
- 有耐心等待极端放量信号
- 追求高盈亏比

**不适合**：
- 无法承受连续 4-5 次亏损
- 追求高胜率
- 短期交易者

## 💡 实战建议

### 1. 与其他策略组合

```
推荐配置：
- v1_plus: 50%（稳健底仓）
- v_kell: 30%（高盈亏比增强）
- v_stine: 20%（避险）

原因：
- v_kell 回撤 29.4%，不能单独使用
- 与低波动策略组合，平衡风险
```

### 2. 耐心等待极端放量

```
Kell 的建议：
"宁愿错过，不要做假突破"
"3 倍放量是我的底线，4-5 倍更理想"

实战：
- 看到突破时不要急着买
- 收盘后确认成交量
- 成交量不足 3 倍，放弃
```

### 3. 接受连续亏损

```
v_kell 胜率 18.2%，意味着：
- 每 5 次交易，4 次亏损
- 连续 10 次亏损是正常的
- 但 1 次大赢可以覆盖所有小亏

心理准备：
- 设置总仓位上限（避免连续止损打击过大）
- 每笔亏损控制在总资金 2% 以内
```
