# O'Neil CANSLIM 技术版策略说明

> **策略ID**: `v_oneil`
> **灵感来源**: William O'Neil《笑傲股市》CANSLIM 系统
> **适用场景**: 成长股牛市，基本面+技术面共振

---

## 📚 策略哲学

**核心理念**："买创新高的成长股，让利润奔跑，快速止损。"

William O'Neil 通过研究 1880 年以来所有牛股，发现一个规律：
- 大牛股在暴涨前都处于"杯柄形态"
- 基本面强劲（EPS、营收高速增长）
- 技术面突破时放量确认

**注意**：我们的技术版缺少 EPS、ROE 筛选（无基本面数据），仅用技术面近似。

---

## 🎯 核心特点（与 v1_plus 的差异）

### 宽松的 VCP（近似杯柄形态）

```
v1_plus: VCP 末端箱体 < 12%
v_oneil: VCP 末端箱体 < 20%  ← 更宽松
```

**O'Neil 的杯柄形态**：
```
杯口 ────┐
         │ 杯深可达 30%
         │（底部）
杯口 ────┤
         │ 柄部回调 5-15%
突破 ─────┘
```

**为什么更宽松？**
- O'Neil 发现：大牛股的整理可以很深（甚至 33%）
- 只要突破时放量，形态就是有效的
- 宽松形态能捕获更多大牛股

---

### 更高的 RS 门槛

```
v1_plus: RS 排名 ≥ 70%（前30%）
v_oneil: RS 排名 ≥ 80%（前20%）← 更严格
```

**IBD 风格 RS 评级**：
- Investor's Business Daily 给每只股票打分
- RS≥80 = 过去12个月表现优于 80% 的股票
- O'Neil：只买 RS≥80 的股票

---

### 更紧的止损

```
v1_plus: 固定止损 10%
v_oneil: 固定止损 7%  ← 更严格
```

**O'Neil 的止损纪律**：
- "亏损超过 7% 就立即卖出，不要犹豫"
- 成长股波动大，小亏损可能快速扩大
- 保护本金是第一原则

---

## 🎯 买入条件

### 1. 趋势确认（同 v1_plus）
- SMA50 > SMA150 > SMA200
- SMA200 上升
- 距 52 周低点 ≥ +25%
- 距 52 周高点 ≤ 25%

### 2. RS 排名 ≥ 80%
- 过去 12 个月涨幅排名前 20%
- 只买市场领涨股

### 3. VCP 形态（宽松）
- 至少 2 次波动收缩
- 末端箱体 < 20%（vs v1_plus 的 12%）
- 量能逐渐萎缩

### 4. 放量突破
- 突破过去 30 日最高价
- 成交量 ≥ 1.5 倍均量

## 🎓 策略优缺点

### ✅ 优点

1. **捕获大牛股**：宽松 VCP 允许深度整理，不错过形态完美的成长股
2. **纪律严格**：7% 止损，保护本金
3. **历史表现优异**：OOS 年化收益 96.1%，夏普 2.62
4. **IBD 实证**：基于 100+ 年牛股研究

### ⚠️ 缺点

1. **缺少基本面**：无 EPS、ROE、营收增长筛选（原版 CANSLIM 需要）
2. **胜率偏低**：37%（vs v1_plus 的 43.7%）
3. **止损频繁**：7% 止损更容易被震出
4. **回撤较大**：最大回撤 18.3%（vs v1_plus 的 15.3%）

## 📈 历史表现

**样本外测试（2024-02-12 至 2026-03-06）**

| 指标 | v_oneil | v1_plus | 对比 |
|------|---------|---------|------|
| 年化收益 | **96.1%** | 54.8% | v_oneil 胜 |
| 夏普比率 | **2.62** | 2.24 | v_oneil 胜 |
| 最大回撤 | 18.3% | 15.3% | v_oneil 略差 |
| 胜率 | 37.0% | 43.7% | v_oneil 略差 |
| 每月信号 | 5.9 | 6.8 | 相近 |
| 达标 | 5/6 | 6/6 | v1_plus 更平衡 |

**结论**：v_oneil 收益更高但波动更大，适合风险承受力强的投资者。

## 💡 适用人群

**适合**：
- 追求高收益（年化 50%+）
- 能承受 20% 回撤
- 严格止损纪律
- 偏好成长股

**不适合**：
- 保守型投资者
- 无法接受频繁止损（7% 容易触发）
- 偏好价值股

## 🔍 与原版 CANSLIM 的差异

| 原版 CANSLIM 要素 | v_oneil 实现 | 状态 |
|------------------|-------------|------|
| C: 当季 EPS 增长 ≥ 25% | ❌ 无数据 | 缺失 |
| A: 年度 EPS 增长 ≥ 25% | ❌ 无数据 | 缺失 |
| N: 新产品/新管理层 | ❌ 无数据 | 缺失 |
| S: 供需关系（小市值优）| ❌ 无数据 | 缺失 |
| L: 领涨股（RS≥80）| ✅ 已实现 | 有 |
| I: 机构买入 | ❌ 无数据 | 缺失 |
| M: 市场趋势 | ✅ 已实现 | 有 |

**重要提示**：v_oneil 仅实现技术面（L+M），缺少基本面筛选，信号质量低于原版。

## 💡 实战建议

### 1. 与 v1_plus 配合使用

```
总仓位分配：
- v1_plus: 60%（稳健底仓）
- v_oneil: 40%（激进增强）

理由：
- v1_plus 提供 6/6 达标的稳定收益
- v_oneil 在牛市中提供超额收益
- 分散策略风险
```

### 2. 止损纪律

```
O'Neil 的建议：
"买入后立即设置止损单，不要抱侥幸心理"
"7% 是绝对红线，跌破立即执行"

实际操作：
1. 买入时设置 GTC 止损单
2. 不手动调整止损价
3. 触发后不后悔，寻找下一个机会
```

```
当前市场状态（2026-03）：
- v_oneil 年化 96%，可能过热

```
