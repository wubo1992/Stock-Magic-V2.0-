## 背景

Phase 2 验证了策略基本有效，但存在以下问题：
1. **信号频率偏低**（每月 < 2 个）：条件过严或股票池太小
2. **无 VCP 形态识别**：初始版只有趋势模板 + 突破，缺少 Minervini 核心的 VCP 模式
3. **无相对强度排名（RS）**：缺少市场领涨股筛选
4. **无信号强度分级**：所有信号无差别输出
5. **无可读报告**：只有 CSV，没有人类可读的日报

---

## 改进内容

### 3.1 VCP 形态识别（Volatility Contraction Pattern）

VCP 是 Minervini 策略的核心特征：
- 在上升趋势中，股价经历多轮回调，每轮幅度递减
- 成交量同步收缩，代表卖盘枯竭
- 末端箱体压缩到 < 8%，类似弹簧被压紧

实现的检测逻辑：
```
识别波动区段 → 计算每段回调幅度 → 验证递减趋势
计算每段均量 → 验证量能收缩
计算末端 10 日箱体宽度 → 验证 < final_range_pct
```

### 3.2 相对强度排名（RS Percentile）

计算股票过去 12 个月涨幅，在当前股票池中排名：
- RS > 70% 才考虑买入
- RS 越高说明该股票是市场领涨者

### 3.3 信号强度评分（1-5 星）

根据触发的条件组合评分：
- 基础（趋势模板 + 突破 + 量能）= ★★★
- 加 RS 高分 = +★
- 加 VCP 形态 = +★
- 满分 5 星 = 所有条件全部通过

### 3.4 Markdown 日报生成（signals/report.py）

每次 live 运行后生成可读报告：
- 今日新增买入信号（含完整条件解读）
- 今日出场信号（止损/追踪止盈/时间止损）
- 过去 7 天信号汇总表
- 过去 7 天历史信号详细解读

## 关键设计决策

| 决策 | 选择 | 原因 |
|------|------|------|
| VCP 最小收缩次数 | 默认 2 次 | 1 次太宽松，3 次信号太少 |
| 末端箱体宽度 | ≤ 8% | Minervini 原著建议 |
| RS 阈值 | ≥ 70% | 筛选前 30% 强势股 |
| 追踪止盈幅度 | 从最高点 -20% | 给利润充足运行空间 |
| 时间止损天数 | 20 个交易日 | 约 1 个月，不动的突破无效 |
| 报告范围 | 过去 7 天 | 覆盖一周交易日，不遗漏信号 |
